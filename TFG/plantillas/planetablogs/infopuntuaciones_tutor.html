Bases de las puntuaciones


  1. Por cada entrada, el usuario que blogee dicha entrada recibirá una bonificación de 8 planets. Si la entrada es eliminada, se penalizará con 10 planets.

  2. Cada valoración positiva que reciba una entrada, corresponde con una bonificación de 2 planets para el usuario que escribió dicha entrada.

  3. Cada valoración negativa que reciba una entrada, corresponde con una penalización de 1 planets para el usuario que escribió dicha entrada.

  4. Cada comentario realizado se premiará con una bonificación de 3 planets para el usuario que escribió dicho comentario. Si el comentario es eliminado se penalizará con 3 planets.
